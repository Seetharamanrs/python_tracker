# Python Expense Tracker (CLI)
    A command-line expense management application built using Python and SQLite.The program allows users to record, manage, and analyse daily expenses.

## Features
- Add, update, and delete expenses (CRUD operations)
- Search expenses by keyword
- Calculate total spending
- Monthly spending summary using SQL aggregation
- Export full report to CSV (Excel compatible)
- Input validation to prevent crashes

## Tech Stack
- Python
- SQLite3
- Pandas

## What I Learned
- Database schema design
- SQL queries (SELECT, INSERT, DELETE, UPDATE, GROUP BY)
- Error handling using try/except
- Working with files and CSV export
- Building a complete CLI application

``` python db_connection.py```
